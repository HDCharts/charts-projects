# Contributing

Thanks for helping improve HDCharts.

## Local Setup

Use this repository to clone the related project repositories into one workspace:

```bash
scripts/clone-repos.sh --all
```

The main library lives in [HDCharts/charts](https://github.com/HDCharts/charts). Documentation and examples live in [HDCharts/charts-docs](https://github.com/HDCharts/charts-docs).

## What To Contribute

Useful contributions include:

- bug fixes
- chart examples
- documentation improvements
- accessibility and API polish
- small, focused feature improvements

For larger changes, open an issue first so the direction can be discussed before implementation.

## Pull Requests

Keep pull requests focused and easy to review:

- describe the user-visible change
- include screenshots or recordings for visual changes
- add or update tests when behavior changes
- update documentation when APIs, examples, or setup steps change

Before opening a pull request, run the relevant checks for the repository you changed.
